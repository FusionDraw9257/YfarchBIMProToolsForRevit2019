# Yfarch BIM Pro Tools 2019

Yfarch BIM Pro Tools 2019 is a toolset for Autodesk Revit 2019. It provides practical BIM automation commands for model production workflows.

## Download

Download the latest installer from the GitHub Releases page:

https://github.com/FusionDraw9257/YfarchBIMProTools2019/releases

Recommended file:

```text
YfarchBIMProTools2019_20260527V1.rar
```

## Supported Environment

- Autodesk Revit 2019
- Windows
- .NET Framework 4.7.2 or compatible environment

## Installation

1. Close Autodesk Revit.
2. Download the latest release package.
3. Extract the `.rar` file if needed.
4. Run the setup executable.
5. Open Autodesk Revit 2019.
6. Use the Yfarch BIM tools from the Revit ribbon.

## Main Tools

- Auto grids
- Auto structural column generation
- Auto structural beam generation
- Auto structural slab generation
- Auto wall join
- Auto floor boundary join
- Column and beam join switch
- Parking space numbering
- Revit to SketchUp export helper
- License information viewer
